# 席決めWebサービス 仕様書

## 概要

懇親会当日の座席をランダムで決めるWebサービス。管理者がテーブル設定と参加者名を事前登録し、振り分けボタンで一斉割り当て。参加者はスマホで自分の名前を入力すると、ガチャ演出付きで自分の席が分かる。

## 技術構成

| 項目 | 内容 |
|------|------|
| フレームワーク | Next.js 16 + TypeScript (App Router) |
| アニメーション | Framer Motion + canvas-confetti |
| データ保存 | Vercel KV（Upstash Redis） |
| デプロイ | Vercel |

## 画面構成

### ユーザーページ (`/`)

1. 名前入力フォーム
2. 「席を確認する！」ボタン
3. ガチャ・ルーレットアニメーション（約3.5秒）
4. 結果表示「X番テーブル」+ 紙吹雪

### 管理者ページ (`/admin`)

1. 参加者リスト入力（テキストエリア、改行区切り）
2. テーブル設定（テーブル数・各定員）
3. 「ランダム振り分け！」ボタン
4. 振り分け結果一覧表示

## API

| Method | Path | 説明 |
|--------|------|------|
| GET | `/api/event` | イベントデータ全体取得 |
| PUT | `/api/event` | 参加者・テーブル設定更新 |
| POST | `/api/assign` | ランダム振り分け実行 |
| GET | `/api/seat?name=xxx` | 特定参加者の席取得 |

## 当日の運用手順

1. `/admin` を開く
2. 参加者名をテキストエリアに入力して「保存」
3. テーブル数・定員を設定して「保存」
4. 懇親会開始時に「ランダム振り分け！」ボタンを押す
5. 参加者は `/` でスマホから自分の名前を入力して席を確認

## デプロイ手順

```bash
# 1. Vercel CLIでログイン
npm i -g vercel && vercel login

# 2. Vercelダッシュボードでプロジェクトを作成
#    → Storage → Create KV → プロジェクトにリンク

# 3. 環境変数を取得
vercel env pull .env.local

# 4. Vercelにデプロイ
vercel --prod
```

## セキュリティ設計

- XSS: ReactのJSX標準エスケープ（`dangerouslySetInnerHTML`不使用）
- APIキー: `.env.local` + Vercel環境変数で管理、ハードコード禁止
- 管理者ページ: 認証なし（URLを知っている人のみアクセス可）
