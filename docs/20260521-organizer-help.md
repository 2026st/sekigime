# 主催者向けヘルプページ

**日付**: 2026-05-21

## 概要

主催者が席決めを初めて使うときの操作手順を `/help` に集約した。トップ・主催者画面からリンクする。

## URL

| ページ | パス |
|--------|------|
| ヘルプ | `/help` |
| トップ（主催者） | `/` |
| 主催者設定 | `/?id={uuid}` |
| 参加者 | `/{uuid}` |

## リンクの使い分け（ページ内でも強調）

- **主催者用** `/?id={uuid}` — 設定・振り分け用。ブックマーク推奨。メンバーに送らない。
- **参加者用** `/{uuid}` — 席作成後に配布。全員同じ URL。

## 変更ファイル

- `src/app/help/page.tsx` — ヘルプ本文
- `src/app/page.tsx` — トップから `/help` へのリンク
- `src/components/OrganizerSetup.tsx` — 主催者画面ヘッダーから `/help` へのリンク

## 主な手順（要約）

1. トップで「席決めを始める」
2. 参加者リストを保存
3. テーブル設定を保存（合計定員 ≥ 参加者数）
4. 「席を作成」
5. 「参加者に配るリンク」をコピーして全員に送信

再開は `/?id=...` のブックマーク、またはトップの「既存の席決めを編集」に参加者 URL を貼る。
