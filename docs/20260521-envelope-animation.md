# 封筒アニメーション（洋形2号・ダイヤ貼）

## 意図

参加者が席番号を確認する体験を、洋形2号（ダイヤモンド貼）の背面形状で表現する。4フラップが背面中央で交差し、上フラップを開けると中から紙が出る。

## 寸法（図面比）

- 基準: 幅 162 × 胴 114、フラップ 69（開時）
- UI スケール（`W = 280`）:
  - `BODY_H = round(280 × 114/162) = 197`
  - `FLAP_H = round(197 × 69/114) = 119`
  - 中央 `(cx, cy) = (140, 98.5)`

|----|------|------|-----|

| 4 | 右 | `(W,0) (W,H) (cx,cy)` | `#b89868` |

上フラップ: `transformOrigin: cx 0`、`rotateX(0 → 180deg)`。SVG は `top: FLAP_H` に配置し、開封時に上方向へ逃がす。

## アニメーション

1. **sealed**: 4フラップ＋シール、紙は非表示
2. **opening**: シール消失 → 上フラップ開く（700ms）→ 紙上昇（delay 350ms）
3. **opened**（1100ms）: 席番号表示、`onOpened()` → Confetti

## 検証

- `/[eventId]` で名前検索 → 未開封でダイヤ貼り＋「席」シール
- タップで上フラップが開き、紙がせり出す
